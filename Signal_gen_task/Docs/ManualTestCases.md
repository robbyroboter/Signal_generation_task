# Тест-кейсы для ручного тестирования приложения

## TC-001 Генерация синусоиды с корректными параметрами

**Предусловия:** приложение запущено.

**Шаги выполнения:**
1. Выбрать тип сигнала `Sine`.
2. Ввести амплитуду `5`.
3. Ввести частоту `2`.
4. Ввести количество точек `1000`.
5. Нажать кнопку `Сгенерировать`.

**Ожидаемый результат:** в таблице отображаются точки сигнала, выводятся максимум, минимум, среднее значение и количество пересечений нуля.

## TC-009 Повторная генерация сигнала с другими параметрами

**Предусловия:** приложение запущено.

**Шаги выполнения:**
1. Сгенерировать сигнал `Sine` с параметрами: амплитуда `5`, частота `2`, точки `1000`.
2. Изменить тип сигнала на `Square`.
3. Изменить амплитуду на `3`.
4. Нажать кнопку `Сгенерировать`.

**Ожидаемый результат:** таблица и результаты обработки обновляются в соответствии с новыми параметрами.
